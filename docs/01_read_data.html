Read in the data

We will first read in single-cell data, previously extracted from processed multi-modal (IMC,IF) images in MapMetIP or downloaded from Zenodo. For reading the data, we use the imcRtools package. This package uses the read_steinbock function to read data into a SpatialExperiment (SPE). The SpatialExperiment class [@Righelli2021] is an extension of the SingleCellExperiment class. For more information, please refer to ?read_steinbock.

Read single-cell measurement

Replace path with the path to the folder where you stored the single-cell data (e.g. path/to/folder/MapMetIP_ProcessedDataset in case processed data was downloaded from Zenodo). In MapMetIP, we exported single-cell marker measurements based on non-dilated and dilated (by 1px) segmentation masks to capture membrane signals. As both are required in the downstream analysis, we will read in both and combine them into one SPE object saving each expression matrix into one assaysslot. Measurements based on non-dilated masks will be in the assaysslot counts and the measurements based on dilated masks will be stored in counts_1px.

path <- file.path(params$input, "MapMetIP_ProcessedDataset")

spe_0px <- read_steinbock(path=path, image_file=NULL, panel_file = file.path(path, 'NB_Panel.csv'), intensities_folder = "intensities-0px",regionprops_folder = "regionprops", graphs_folder=NULL) 
spe_1px <- read_steinbock(path=path, image_file=NULL, panel_file = file.path(path, 'NB_Panel.csv'), intensities_folder = "intensities-1px",regionprops_folder = "regionprops", graphs_folder=NULL) 

#Combine SPE objects into one object
spe <- spe_0px
assay(spe, "counts_1px") <- assay(spe_1px, "counts")

#remove single SPE objects
rm(spe_0px)
rm(spe_1px)

Read single-cell metadata

We set the colnames of the object to generate unique identifiers per cell.

colnames(spe) <- paste0(spe$sample_id, "_", spe$ObjectNumber)

When then read sample and patient-specific metadata from the metadata file provided on Zenodo (Supplementary_Tables/20240808_Suppl_Table1.xlsx).

file_path <- file.path(params$input, 'Supplementary_Tables/20240808_Suppl_Table1.xlsx')
sample_spec_metadata <- read_excel(file_path, sheet = "sample_spec_metadata")
patient_spec_clinical <- read_excel(file_path, sheet = "patient_spec_clinical")
patient_spec_genetic <- read_excel(file_path, sheet = "patient_spec_genetic")

# Merge patient and sample-specific data
merged_data <- sample_spec_metadata %>%
  left_join(patient_spec_clinical, by = "Patient_ID")

meta <- merged_data %>%
  left_join(patient_spec_genetic, by = "Patient_ID")

and assign it to the colData slot in our SPE object.
